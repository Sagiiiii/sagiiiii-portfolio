markdown# 🦊 Sagiii.dev — Portfolio Personal

Portfolio personal desarrollado con Python y Flask, tema oscuro con colores fox/naranja.

## 🚀 Stack
- **Backend:** Python + Flask
- **Frontend:** Bootstrap 5 Dark + CSS personalizado
- **Email:** Flask-Mail + Gmail
- **Deploy:** Render.com

## 📄 Páginas
- **Inicio** — Hero, tecnologías y proyectos destacados
- **Proyectos** — Galería con filtros por categoría
- **Sobre mí** — CV, experiencia y formación
- **Contacto** — Formulario con envío a Gmail

## ⚙️ Instalación local
```bash
git clone https://github.com/Sagiiiii/portfolio.git
cd portfolio
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python run.py
```
